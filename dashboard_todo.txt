Képmódosítás:
    - Az input group bodyjába kártyákba (pici), hoverre két gomb megjelenik:
        - Ceruza: Új kép kiválasztása (file input, amikor benyomja, akkor kicseréli a kártya URLjét)
        - Kuka: Kép törlése (megerősítő üzenet, majd kitörli a kártyát)
    - Plusz ikonos gomb, amivel újat lehet felvinni, ahol több is lehet (termékképek)

    - Hidden input a módosítás gomb fölé:
        - JS-sel követjük a szükséges módosításokat:
            - Kép típusa + url : Művelet (DELETE, CHANGE, ADD)
        - Form beadásnál belerakjuk JSON kódolva a rejtett mezőbe
        - PHP végigmegy a módosításokon (foreach), és optimalizálja az új képeket, adatbázisban nem kell módosítani, csak ha újat tölt fel.
    
    - Ha nincs képe egy terméknek, akkor egy alap jöjjön be, de ez csak az SQL lekérdezések változtatását jelenti.
